# **App Name**: KwanzaKeeper

## Core Features:

- Expense Logging & Management: Users can log new expenses with comprehensive details including amount, category, description, date, payment method, payer, and notes, as well as easily view and delete existing entries.
- Categorization & Quick Add Shortcuts: Efficiently categorize expenses using a set of predefined labels with associated emojis and utilize quick-add buttons for frequently recorded transactions to speed up data entry.
- Monthly Financial Overview: Provides an intuitive dashboard summarizing monthly spending totals, daily average expenditure, and detailed breakdowns of costs by category and individual payers.
- Dynamic Search & Filtering: Allows users to quickly find specific transactions by applying filters based on keywords, expense categories, or specific dates.
- Secure Data Persistence: Securely stores and retrieves all user expense data in a cloud-based database, ensuring data integrity and seamless access across sessions and devices.
- User Authentication: Enables users to create and manage personal accounts, securing their financial data and providing a personalized experience.
- AI-powered Spending Insights Tool: A generative AI tool that analyzes spending patterns to offer personalized insights, identify potential savings, and flag unusual or excessive expenditures.

## Style Guidelines:

- Primary Color: A vibrant, energetic orange (#FF7E08), chosen to represent financial clarity and active management against the dark theme. This hue provides strong contrast and signals key interactions.
- Background Color: A very dark, almost black, with a subtle warm undertone (#151211). This deep, muted hue prevents eye strain in low-light environments and allows primary and accent colors to pop, creating a sophisticated and focused interface.
- Accent Color: A soft, muted coral (#E87A81), analogous to the primary color but with different saturation and brightness. This color is used for secondary calls to action or to highlight less critical information, providing visual balance.
- Headlines and body text: 'Sora' (sans-serif) for its modern and clean appearance. Numeric data and amounts: 'DM Mono' (monospace) for clear, consistent display of financial figures. Note: currently only Google Fonts are supported.
- Utilize expressive and diverse emojis for quick categorization and visual identification of expenses. Supplement with a minimalist set of line-based icons for navigation and functional UI elements, maintaining consistency and clarity.
- Implement a clean, mobile-first design with a clear hierarchy using card-based elements for expense listings and summary sections. Maintain consistent padding and margins for readability, anchored by a fixed bottom navigation for easy access to main views.
- Incorporate subtle and functional animations, such as gentle slide-ins for new views, seamless fades for content updates, and discrete micro-interactions on button presses, to enhance user feedback and fluid navigation.
